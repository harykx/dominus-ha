# Immergas Home Assistant Add-on Repository

This repository contains a Home Assistant add-on for the Immergas Magic Combo Plus V2 local TCP module.

## Install

In Home Assistant:
1. Go to **Settings -> Apps**
2. Open the overflow menu
3. Choose **Repositories**
4. Add your Git repository URL
5. Install **Immergas Bridge**

## Configure

Set:
- heater IP
- heater MAC
- heater password
- MQTT broker host and credentials

For the official Mosquitto add-on, use `core-mosquitto` as the MQTT host.
